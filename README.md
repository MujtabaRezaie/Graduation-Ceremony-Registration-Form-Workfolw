# Graduation-Ceremony-Registration-Form-Workfolw (n8n)
This workflow collects and manages student registrations for a university graduation ceremony using a form, Google Sheets, and email.

Users fill out a form with their personal and academic information, including name, field of study, university, graduation year, and email. Once the form is submitted, the data is automatically stored in Google Sheets. If a record with the same email already exists, it is updated.

At the same time, a confirmation email is sent to the user to notify them that their registration was successful.

The workflow supports Persian form input and ensures organized data collection for event management.

Requirements include n8n, Google Sheets API, and Gmail API.
